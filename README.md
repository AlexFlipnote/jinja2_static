# jinja2_static
Converts Flask/Jinja2 template files to static files.

## Requirements
- Python 3.6 or above

## Supported files
- .jinja
- .html
- .md

## Docs
Not documented yet, but I added stuff to `/tests/` folder.

## Install
It's not on `PyPi` so use this:<br>
pip install git+https://github.com/AlexFlipnote/jinja2_static
